### 이어 붙인 수 😀

양의 정수 n이 매개변수로 주어질 때, <br>
n이 홀수라면 n 이하의 홀수인 모든 양의 정수의 합을 return 하고 <br>
n이 짝수라면 n 이하의 짝수인 모든 양의 정수의 제곱의 합을 return 하는 solution 함수를 작성해 주세요<br>

※ 제한 사항<br>
1 ≤ n ≤ 100<br>

입출력 <br>
n: 7<br>
result: 16<br>

n: 10<br>
result: 220<br>

```javascript
function solution(a, b, flag) {
    if(flag === true) {
        return (a+b);
    }else {
        return(a-b);
    }
}
```
